  <h1 align="center">Telegram Attach Bot</h1>
</p>

A bot to attach a media to a long message. Written using "Pyrogram" wrapper for TelegramBots

## Installation
- Create a new bot using @BotFather. Add the bot to channel as admin
- Edit config.py
- run
